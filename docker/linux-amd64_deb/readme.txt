
Установка из графического интерфейса
____________________________________

Распакуйте архив.
Если ваш менеджер файлов позволяет запускать ярлыки (desktop-файлы),
запустите ярлык установщика.
Если ваш менеджер файлов позволяет запускать скрипты,
запустите install_gui.sh.
Он сам попросит повышения привилегий.
Для его работы необходим пакет whiptail/newt.


Установка из консоли
____________________

Распакуйте архив.
Перейдите в каталог с дистрибутивом.
Запустите скрипт install_gui.sh c псевдографическим интерфейсом.
Он сам попросит повышения привилегий.
Для его работы необходим пакет whiptail/newt.


Выбор нужных компонентов
________________________

Никогда не устанавливайте пакеты без необходимости:
это усложнит настройку и может снизить производительность.
Если вы не знаете точно, что вам нужно, оставляйте набор по умолчанию.


Обновление (переустановка)
__________________________

Начало аналогично установке.
В псевдографическом интерфейсе выберите "Переустановить" ("Reinstall").
Ключи и сертификаты пользователя будут сохранены.
Системные настройки провайдера будут сброшены.


Удаление
________

Начало аналогично установке.
В псевдографическом интерфейсе выберите "Удалить" ("Uninstall").
Ключи и сертификаты пользователя будут сохранены.
